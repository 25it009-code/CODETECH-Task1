# Personal Finance Tracker – Project Overview

## Introduction

The **Personal Finance Tracker** is a web-based application used to manage personal income and expenses.
It helps users track their financial activities, calculate total balance, and monitor spending habits easily.

This project is developed using:

* HTML
* CSS
* JavaScript

The application provides a simple and user-friendly interface for maintaining daily financial records.

---

# Objectives

* To track income and expenses
* To calculate total balance automatically
* To improve money management
* To provide a simple financial monitoring system
* To learn front-end web development concepts

---

# Features

## 1. Add Transactions

Users can add:

* Income
* Expenses

with:

* Description
* Amount
* Transaction Type

---

## 2. Balance Calculation

The application automatically calculates:

* Total Income
* Total Expense
* Remaining Balance

---

## 3. Delete Transactions

Users can remove unwanted transactions using the delete button.

---

## 4. User-Friendly Interface

The project has:

* Clean design
* Responsive layout
* Easy navigation

---

# Technologies Used

| Technology | Purpose                  |

# Working Process

1. User enters transaction details
2. Data is stored temporarily in JavaScript array
3. Application updates:

   * Income
   * Expense
   * Balance
4. Transaction list is displayed dynamically
5. User can delete transactions anytime
